How to Install Junior on Windows

Quick guide to get past Windows SmartScreen and install Junior safely

⚠️ Windows SmartScreen Notice

Junior is currently awaiting Windows code-signing certification. Until that process is complete, Windows SmartScreen will show a "Windows protected your PC" warning when you run the installer. This is normal and expected. The app is safe to install — follow the steps below.

📥 Installation Steps

Step 1: Download the Installer

  1. Download Junior for Windows from your purchase confirmation page
  2. Locate the downloaded .exe file (usually in your Downloads folder)
  3. Double-click the installer to begin

Step 2: You'll See This Warning ⚠️

Windows protected your PC

Microsoft Defender SmartScreen prevented an unrecognized app from starting.

Don't worry! This warning appears for any app that hasn't yet been certified by Microsoft. It does not mean the app is harmful. Follow Step 3 to continue.

Step 3: Click "More info" → "Run anyway"

✅ Two clicks to install

  1. On the SmartScreen warning, click "More info" (small text link below the warning message)
  2. A "Run anyway" button will appear — click it
  3. If a User Account Control (UAC) prompt appears asking "Do you want to allow this app to make changes?", click "Yes"
  4. The Junior installer will launch — follow the on-screen steps to complete installation

📌 Important

  • You only need to do this once — during the initial install
  • After installation, Junior will open normally without any warnings
  • We are actively completing the Windows code-signing certification process
  • Once certified, this SmartScreen warning will no longer appear for new installs

❓ Frequently Asked Questions

Is Junior safe to install?

Yes. Junior is safe and does not contain malware. The SmartScreen warning appears because we are an independent developer and our Windows code-signing certificate is currently being processed. This is the same warning that appears for thousands of legitimate apps from small developers.

Why isn't Junior code-signed for Windows yet?

Windows code-signing certification requires a multi-step verification process with Microsoft. We are currently going through this process. Once complete, SmartScreen will recognize Junior as a trusted application and the warning will no longer appear. Our macOS version is already verified with Apple.

What if I don't see the "More info" link?

If your organization's IT policy blocks the "Run anyway" option, try the following:

  1. Right-click the downloaded .exe file
  2. Select Properties
  3. At the bottom of the General tab, check the "Unblock" checkbox
  4. Click Apply, then OK
  5. Double-click the installer again
