Windows Install Help

Install Junior on Windows safely in minutes.

You may see SmartScreen while Windows certification is in progress. That warning is expected and harmless for this installer.

See Steps Go to First Launch Guide

SmartScreen notice

Junior is safe to install. Windows may display "Windows protected your PC" because Microsoft code-signing certification is currently in progress. This warning is expected and harmless in this context.

Installation steps

Step 1: Download and launch the installer

  1. Download Junior for Windows from your confirmation page.
  2. Open your Downloads folder and locate the .exe file.
  3. Double-click the installer.

Step 2: If SmartScreen appears

You may see: "Windows protected your PC"

This does not indicate malware for this installer. Continue with step 3.

Step 3: Click "More info" and continue

  1. Click More info on the SmartScreen prompt.
  2. Click Run anyway.
  3. If User Account Control appears, click Yes.
  4. Complete the installer steps on screen.

FAQ

Why does this warning appear?

Because Windows reputation/certificate validation is still in progress. That process takes time for new software releases.

Will this warning disappear later?

Yes. As the code-signing process completes and reputation builds, new installs should see fewer SmartScreen interruptions.
