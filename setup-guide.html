First Launch Setup

Set up Junior once, then run with confidence.

This one-time flow opens Chrome, completes LinkedIn authentication, and saves your session so future runs are faster.

See Setup Steps See Strategy Page

Important before starting

Always enable the 2FA checkbox on your first run. Even without explicit 2FA enabled, this makes the login handoff more reliable.

Before you start

  • Install Junior first (Windows guide / macOS guide).
  • Have your LinkedIn credentials ready.
  • Keep your 2FA device available if LinkedIn prompts for it.
  • Confirm Google Chrome is installed.

Step-by-step first run

Step 1: Launch Junior

Open Junior from your desktop, Start menu, or Applications folder.

Step 2: Check the 2FA box

Enable the 2FA checkbox before clicking start for a clean login handoff.

Step 3: Click Start Automation

Junior opens Chrome automatically.

Step 4: Log in to LinkedIn

Complete login normally in the opened Chrome window.

Step 5: Let Junior close the window

Junior closes Chrome when session capture succeeds. Do not close it manually.

FAQ

Do I need to log in every time?

No. This process is one-time per device unless your LinkedIn session expires or credentials change.

Does Junior save my password?

No. Junior stores session data, not your LinkedIn password.
